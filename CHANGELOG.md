# Changelog

All notable changes to "Timely Chat" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.2] - 2025-01-16

### Added
- Chat history management (save, view, export, clear)
- Code selection sharing to chat (`Ctrl+Alt+S`)
- Status bar integration with connection status
- Context menu for sending selected code
- Multiple keyboard shortcuts

### Changed
- Improved error handling with retry mechanism
- Better token management with 24-hour caching

### Fixed
- Token refresh on configuration change

## [0.0.1] - 2025-01-15

### Added
- Initial release
- Sidebar chat panel
- Editor chat panel
- Timely GPT AI integration
- Basic authentication with Timely API
- Configuration options for API key, Space ID, and user settings
